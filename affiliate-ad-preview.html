Affiliate Ad Preview

Installation Instructions

Only thing that is required is affiliateID. KZ NEEDS To replace url with CM.com before WE LAUNCH THE PROGRAM WITH NEW THINGS.

Ad Code

<script src="/cmaffads.js"></script>
<div id="cmAffiliateAd0"
	data-affiliateid="YOURAFFILIATEID"
></div>

Default View

If no parameters are supplied defaults are used (campaign=000, width=300, height=250).

Available Parameters

  • data-width - numeric - ex: 250
  • data-height - numeric - ex: 250
  • data-affiliateid - string - ex: aff123
  • data-campaign - 3 digit number - ex: 001

Available Sizes

  • 300x250
  • 336x280
  • 728x90
  • 300x600
  • 250x250
  • 468x60
  • 240x400

Ad Sizes

Note: Must have ad slots in sequential order. EX if you have 0, 1, 3 - 3 will not load. Does not need to be in order in the html.

Affiliate Ad Preview

Installation Instructions

Only thing that is required is affiliateID. NEED To replace url with CM.com before launch.

Ad Code

<script src="{{ site.baseurl }}/js/cm-affiliate-ad.js"></script>
<div id="cmAffiliateAd0"
	data-affiliateid="YOURAFFILIATEID"
></div>

Default View

If no parameters are supplied defaults are used (campaign=000, width=250, height=250).

Available Parameters

  • data-width - numeric - ex: 250
  • data-height - numeric - ex: 250
  • data-affiliateid - string - ex: aff123
  • data-campaign - 3 digit number - ex: 001

Available Sizes

  • 250x250
  • 468x60
  • 728x90
  • 240x400

Ad Sizes

Note: Must have ad slots in sequential order. EX if you have 0, 1, 3 - 3 will not load. Does not need to be in order in the html.

250x250

468x60

728x90

240x400

Additional States

If no affiliateid is passed, error appears.

Simple Ad Instructions

The script should only be added once per page. Replace everything in orange for new ads. Iterate the number after cmSimpleAd if there are multiple ads on one page. Ad will be responsive if no data-width or data-height is supplied. Ad image should be 2x the expected display size to be optimized for retina screens.

Ad Code

<script src="http://downloads.channelmaster.com/cmsimpleads.js"></script>
		<div id="cmSimpleAd0"
			data-imagepath="http://downloads.channelmaster.com/Banner+Images/category-banner-outdoor-001.png"
			data-url="http://www.channelmaster.com/ProductDetails.asp?ProductCode=CM-7500GB16"
			data-campaign="cat-outdoor-antenna"
		></div>

Available Parameters

  • data-imagepath - fullurl - ex: http://downloads.channelmaster.com/Banner+Images/category-banner-outdoor-001.png
  • data-url - fullurl - ex: http://www.channelmaster.com/ProductDetails.asp?ProductCode=CM-7500GB16
  • data-campaign - string - ex: cat-outdoor-antenna
  • data-width - numeric - ex: 250 (leave off for responsive ad)
  • data-height - numeric - ex: 250 (leave off for responsive ad)

Additional States

If no imagepath is passed, error appears.
